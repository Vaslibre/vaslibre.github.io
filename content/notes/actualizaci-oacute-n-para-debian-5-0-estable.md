Debian Lenny se distribuye en:  
CD: Colección de 31 CD con la totalidad de los paquetes oficiales disponibles. Los paquete se ordenan según la popularidad, por lo que se estima que los tres primeros CD cumplirían las necesidades de un sistema estándar. En las últimas versiones, se han hecho disponibles, sabores del primer CD con el entorno de escritorio completo para los desktops KDE y XFCE (+LXDE). El CD default, trae consigo GNOME.  
DVD: Colección de cinco DVD con la totalidad de los paquetes oficiales disponibles. Se estima que con el primer DVD, se cumpliría la instalación de un sistema Debian estándar.  
Netinstall: Trae solo lo estrictamente necesario para instalar Debian a través de internet.  
Businesscard: Lo mismo que netinstall, pero de menor tamaño (<40 Mb)
