[Inkscape](http://es.wikipedia.org/wiki/Inkscape) es una herramienta de dibujo libre y multiplataforma para gráficos vectoriales SVG. Este programa surgió de una bifurcación del proyecto Sodipodi. Las características de SVG soportadas incluyen formas básicas, trayectorias, texto, canal alfa, transformaciones, gradientes, edición de nodos, exportación de SVG a PNG, agrupación de elementos, etc.

La principal motivación de Inkscape es proporcionar a la comunidad una herramienta de código abierto de dibujo SVG que cumpla completamente con los estándares XML, SVG y CSS2.

Inkscape se encuentra desarrollado principalmente para el sistema operativo GNU/Linux, pero es una herramienta multiplataforma que funciona en Windows, Mac OS X, y otros sistemas derivados de Unix.

[![Inkscape](http://cursorlibre.com/wp-content/uploads/2010/08/inkscape-048.jpg)](http://www.inkscape.org/)

Sus Principales caracteristicas.

> Capacidad para seleccionar varios trazos al mismo tiempo y editarlos  
> Mejoras en las opciones para la edición de textos  
> Exportación a Latex y PDF  
> Mejores caracteristicas en el uso de mascaras y clips  
> Nueva herramienta spray.  
> Mejoras en la interfaz de usuario  
> Entre otras…

Descarga de paquetes en [sourceforge](http://sourceforge.net/projects/inkscape/files/)

Instalar desde PPA en ubuntu Lucyd 10.04

_sudo add-apt-repository ppa:ricotz/ppa_

_sudo apt-get update_

_sudo apt-get install inkscape_
