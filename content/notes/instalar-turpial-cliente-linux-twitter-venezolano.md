Su instalación es bastante sencilla, solo debes realizar lo siguiente:

![Logo Turpial](http://turpial.org.ve/wp-content/themes/turpial-2.0/images/footer-logo.png)  
Abrir consola y escribir (**ubuntu y derivado**s):

_**sudo add-apt-repository ppa:effie-jayx/turpial  
sudo apt-get update  
sudo apt-get install turpial**_

**.- Fedora**

Como root  
  
_yum install turpial_

**.- Debian**

Como root  
  
_aptitude install turpial_

**.- ArchLinux (AUR)**

_yaourt -S turpial_
