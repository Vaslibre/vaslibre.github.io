Su base es el código base de Firefox 3.6 dandole mayor velocidad de respuesta.

Usa Gecko 1.9.2, para mejorar el rendimiento, la estabilidad, y compatibilidad web

Le ha sido añadido un nuevo asistente de migración y un nuevo administrador de archivos guardados.

Una nueva barra de filtrado rápido. A parte de poder buscar como en Thunderbird 3 desde la barra de búsqueda, ahora se permite filtrar rápidamente correos según diferentes parámetros: si lo tenemos marcado, si tiene adjuntos, si está sin leer, etc.
