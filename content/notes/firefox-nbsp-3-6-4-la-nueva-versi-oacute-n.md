Firefox 3.6.4 proporciona una navegación sin interrupciones para los usuarios de Windows y Linux aunque fallen los plugins Adobe Flash, Apple Quicktime o Microsoft Silverlight.

[![Firefox](http://www.mozilla-europe.org/img/tignish/home/feature-logo.png)](http://www.mozilla-europe.org/es/)

Si un plugin falla o se congela, no afectará al resto de Firefox. Podrás recargar la página para reiniciar el plugin e intentarlo de nuevo.

Firefox 3.6.4 también corrige los siguientes problemas presentes en versiones anteriores a Firefox 3.6

*   Varios [problemas de seguridad](http://www.mozilla.org/security/known-vulnerabilities/firefox36.html#firefox3.6.4).
*   Diversos problemas de estabilidad.
*   [lista completa de cambios](https://bugzilla.mozilla.org/buglist.cgi?quicksearch=ALL%20status1.9.2%3A.4-fixed)
