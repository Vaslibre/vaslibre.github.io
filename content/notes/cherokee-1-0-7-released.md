Cherokee es un servidor web muy rápido, flexible y fácil de configurar.

Es compatible con las tecnologías de hoy en día: FastCGI, SCGI, PHP, CGI, TLS y SSL conexiones cifradas, los hosts virtuales, autentificación, compatibilidad con logs de apache, balanceo de carga HTTP, Reverse HTTP Proxy, Bandwidth Shaping, Audio and Video Streaming, uWSGI, Data Base Balancing  y mucho más

![Cherokee](http://www.cherokee-project.com/images/cherokee2-mini.png)

  
Cherokee también ofrece una interfaz web fácil de facil uso. Permite configurar el servidor de arriba a abajo sin tener que editar un archivo de configuración de texto. "en la mayoria de las veces"
