![Xanadu Linux](http://xombra.com/images/publicaciones/xanadu2.jpg)

  

Xanadu está basada en la versión de desarrollo de Debian de nombre clave SID. Xanadu se puede considerar un derivado de Debian, ya que cumple con las directrices y lineamientos estipulados por Debian.

Xanadu emplea el escritorio LXDE, que ofrece un entorno de escritorio rápido, ligero y eficiente, igualmente contiene herramientas para rescate de sistemas, útiles programas para realizar test de análisis forense y lo que llama verdaderamente la atención de esta distribución es la inclusión de navegación anónima usando bien sea TOR o i2p.

Además trae consigo todos los programas necesarios para un usuario final, como por ejemplo:

LibreOffice (_Herramienta de Oficina_)  
Evince (_Lector PDF_)  
GIMP (_Editor de Imagenes_)  
Brasero (_Quemador_)  
VLC (_Reproductor de Audio y Vídeo_)  
Icewasel _(Navegador Libre basado en Firefox_)  
Thunar (_Gestor de Archivos_)

Xanadu lleva incluidos una gran cantidad de Drivers (_libres y no libres_) para que cualquier dispositivo que sea conectado al equipo sea reconocido de inmediato. Claro, habrá sus excepciones pero no son muchos.

*   La última versión de Xanadu es la 0.6.0 trae las siguientes características resaltantes:
*   Usa kernel liquorix.
*   Usa FISH como shell por defecto.
*   Agregado soporte para XFS y BTRFS.
*   Incorporado apparmor.

Los requerimientos mínimos para su instalación:

*   512 Mb RAM
*   CPU a 800 Mhz
*   Tarjeta de vídeo a 800x600, 16 Mb vídeo RAM
*   BIOS con capacidad de arranque por USB o CDROM
*   5 Gb de capacidad de disco para instalar
