Se había planificado que saliera para el mes de septiembre (finales) pero no fue así ya que se estaban realizando debates sobre el estado del proyecto y se decidió el lanzamiento seis meses. En este caso estaríamos hablando que para marzo/abril del 2011 la 3.0 de Gnome este liberada.  
  
Gnome 2.9.1, incluye varios módulos que aún tienen que ser portados a la nueva API libnotifyy que algunos módulos “ni siquiera funcionan en este momento”

Solo nos queda esperar hasta el año proximo para tener Gnome 3.0
