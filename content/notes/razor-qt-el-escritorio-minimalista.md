Según indican trae como nuevo: un editor de políticas de seguridad (_razor-policykit_), una herramienta de administración de energía (_razor-power_) y una herramienta para gestionar las acciones que el computador realiza (_apagar, suspender_) al cerrar la tapa, llamada razor-autosuspend.

Para instalarlo en Ubunu/LinuxMint y derivados basta con hacer lo siguiente:

> _**sudo add-apt-repository ppa:razor-qt/ppa  
> sudo apt-get update  
> sudo apt-get install razorqt**_

Por alguna razón desconocida o.O no instala el config, pero se soluciona facilmente

> _**sudo apt-get install razorqt-config**_  

Reinicia y elige Razor ;)

Para otras distros visitar el sitio oficial:

[http://razor-qt.org/install/](http://razor-qt.org/install/ "Para otras distros visitar el sitio oficia")
