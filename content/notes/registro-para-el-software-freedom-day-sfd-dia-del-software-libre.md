El Día de la Libertad de Software (Software Freedom Day en inglés) es una celebración anual a nivel mundial del software libre. Es un evento de educación pública, no solo para celebrar las virtudes de este tipo de software, sino para promover su uso para el beneficio del público.

La página de registro para el evento: [http://vaslibre.org.ve/registros/](http://vaslibre.org.ve/registros/ "Registrate!")  

[![Se celebrara el SFD - Septiembre 21!](http://wiki.softwarefreedomday.org/Promote?action=AttachFile&do=get&target=SDF-Asistire.png)](http://vaslibre.org.ve/registros/)

Para el SFD el tema principal serán los productos de Mozilla. Entre los temas que se presentaran ese día están:

\- Presentación FirefoxOS (_Que es y por que es diferente_)  
\- Aprende a desarrollar aplicaciones para FirefoxOS en 15 minutos (_desarrollo_)  
\- Documentación para Desarrolladores la MDN y como colaborar (_localización_)  
\- Guía de Usuario de FirefoxOS aprende a usar todo el potencial del teléfono. (_demos_)  
\- Soporte para FirefoxOS  
\- Como colaborar con Firefox  
\- Ejemplo de creación de Theme para Mozilla Firefox  
\- Ejemplo de creación de Complementos (_Addons_) Mozilla Firefox  
\- Uso de Firefox Sync  
\- Tunning Mozilla Firefox  
\- Securizando Mozilla Firefox (_Usando Complementos como WOT_)  
\- Uso de Firebug para depurar sitios web  
\- Evitar que te rastreen (_Do not track me_)  
\- El Navegador Firefox en tu teléfono Android  
\- Thunderbird el mejor cliente de correo
