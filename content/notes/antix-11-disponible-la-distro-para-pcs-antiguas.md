antiX es una distribución de linux en live CD rápida, ligera y fácil de instalar basada en Debian Testing y MEPIS para equipos basados en Intel-AMD x86. antiX resultará muy cómodo para los usuarios que vienen de Mepis, pero usando un entorno apropiado para computadoras más viejas.

El objetivo de antiX es proporcionar un sistema operativo libre ligero pero totalmente funcional y flexible tanto para novatos como para usuarios experimentados de Linux. Puede usarse desde una vieja PII 266 con 64MB y 128MB de swap hasta cualquier computadora actual, mucho más potente. Se recomienda 128MB de RAM. También se puede usar antiX como un cd de recuperación con un arranque rápido.
