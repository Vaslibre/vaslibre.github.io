Claro que el tiempo ha pasado y hoy Turpial nos ofrece una gran cantidad de opciones, como el amplio soporte para las API de Twitter para poder enviar respuestas, retweets (con el nuevo formato o a través del más tradicional RT), menciones, mensajes directos y demás, y una excelente integración en el escritorio. Respecto a esto último incluso se comenta la posibilidad de que sea incorporado como el cliente de microblogging por defecto para Xubuntu.

También podremos silenciar temporalmente a aquellos usuarios que suelen escribir demasiado y en los cuales puede que momentáneamente no estemos interesados, para que no llenen nuestro timeline, y contamos con la posibilidad de seleccionar varios servicios para el acortamiento de URL y para subir imágenes.

![Turpial Twitter](http://turpial.org.ve/wp-content/uploads/2010/04/turpial-light.png)

Por último, para los fanáticos de la información (o de Twitter en general) se puede utilizar su modo extendido para visualizar tres columnas donde podremos tener abundante información (una para nuestro perfil, otra para alguna búsqueda y otra para los RT por ejemplo).

Para instalar Turpial en Ubuntu:

_**add-apt-repository ppa:effie-jayx/turpial**_

Luego actualizamos la lista de repositorios:

_**sudo apt-get update**_

Y por último:

_**sudo apt-get install turpial**_

Más información: [Turpial](http://turpial.org.ve/)
