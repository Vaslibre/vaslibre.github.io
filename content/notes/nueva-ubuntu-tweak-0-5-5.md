Ubuntu Tweak se ha vuelto ya imprescindible en nuestro Ubuntu, pues nos permite personalizarlo a golpe de ratón de una forma sencilla. Muchos votaríamos por que se incluyera de serie en próximas ediciones de Ubuntu.

En esta versión se añaden las siguientes novedades:

> Soporte para Ubuntu 10.10 Maverick  
> Administrador de ficheros ahora soporta la edición de múltiples tipos de ficheros a la vez, ¡gracias a Muzuiget!  
> Soporte añadido para Lubuntu y Netbook Edition  
> Añadido soporte para iconos SVG GNOME (LP: #593513)  
> Añadido soporte SVG en las preferencias para el logo de la ventana de entrada  
> Añadido script para compresión de ficheros PDF  
> Añadido soporte para eliminar repositorios PPA en Cleaner

Si es la primera vez que lo instalas sigue los siguientes pasos:

_sudo add-apt-repository ppa:tualatrix/ppa_  
_sudo apt-get update && sudo apt-get install ubuntu-tweak_

En caso de ya tenerlo instalado, es suficiente con ejecutar "_sudo apt-get upgrade_" desde terminal.
