El sistema operativo Linux es uno de los más sofisticados y que requiere de conocimiento previo para la utilización del mismo, puesto que es la única herramienta donde el usuario elije y arma su escritorio a como prefiera, sin correr riesgo de nada.

Bastille, una disposición de software para los usuarios que requieren más seguridad, no es solo un programa propiamente dicho de seguridad, sino más buen un conjunto de scripts ideados para dotar de más seguridad nuestra distribución GNU/Linux. De esta forma ayuda a proteger la vulnerabilidad del sistema, optimizando el sistema basándose en las fuentes de Security Linux protegiendo el núcleo de ataques malintencionados.

Si bien Linux es seguro, Bastille logra aún más seguridad dentro de los usuarios, evitando posibles agujeros de seguridad como algunos ya conocidos. Esto lo puede hacer cualquier usuario avanzado ya que como son scripts conocidos, permite que este conjunto de herramientas estén al alcance de todos, independientemente de sus conocimientos. Su instalación y configuración no es complicada, está diseñada para que los usuarios puedan utilizarla de manera didáctica ya que los más nuevos entenderán un poco mejor como funciona su sistema.
