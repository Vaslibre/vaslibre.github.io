**Kompozer:** Es un editor HTML WYSIWYG ( What You See Is What You Get (en inglés, "lo que ves es lo que obtienes")) basado en su antecesor NVU; es mantenido por la misma comunidad de usuarios, muy completo y sencillo de usar. Esta orientado a personas que no dominan muy bien el HTML pero que desean comenzar a realizar poryectos web.

![Kompozer](http://www.awven.com/salida.php?imagen=http://kompozer.sourceforge.net/screenshots/kpz-scr-03.png)

Sitio web: [http://www.kompozer.net/](http://www.kompozer.net/)

**BlueGriffon**: Entra recientemente en el ámbito del diseño web, es sencillo y poderoso, emplea el motor Gecko el mismo que usa Firefox. Esta creado para soportar HTML5 y CSS3.

![BlueGriffon](http://www.awven.com/salida.php?imagen=http://bluegriffon.org/public/info/mainwindow-shot.png)

Sitio web: [http://www.bluegriffon.org/](http://www.bluegriffon.org/)

**BlueFish**: Utiliza un editor GTK + HTML. Incluye varios asistentes que ayudan en el diseño. Implementa también un completo cuadro de diálogo de ayuda para la inserción de imágenes. Permite la inserción de tumbnails (imágenes en miniatura) y enlaces de acceso a la imagen original. Orientado a programadores.

![BlueFish](http://www.awven.com/salida.php?imagen=http://bluefish.openoffice.nl/screenshots/mac_bookmarks.png)

Sitio web: [http://bluefish.openoffice.nl/](http://bluefish.openoffice.nl/)

**Aptana**: Uno de los más usados en el ambinte de Software libre. Aptana utiliza un IDE centrado en JavaScript, pensado para el diseño dinámico de sitios web. Implementa un asistente de código para JavaScript, HTML y CSS, en el que se incluyen también varias funciones predefinidas en JavaScript.

Entre sus características se pueden mencionar:  
\- Asistente de código para HTML y Javascript.  
\- Librerías ajax (jQuery, prototype, scriptaculous, Ext JS, dojo, YUI y Spry entre otras).  
\- Conexión vía FTP, SFTP, FTPS y Aptana Cloud.  
\- Herramientas para trabajo con base de datos.  
\- Marcado de sintaxis mediante colores.  
\- Compatible con extensiones para Eclipse (existen +1000).

![Aptana](http://www.awven.com/salida.php?imagen=http://www.aptana.com/images/product/S2-preview-lrg.png)

Sitio web: [http://www.aptana.com/](http://www.aptana.com/)

**Amaya:** Es un editor WYSIWYG que respeta, casi al 100 por 100, las últimas propuestas de la W3C está compuesta por un navegador web y una herramienta de autor. Cualquier página web que se abra puede ser editada inmediatamente. Incluye además XHTML, soporte para SVG (Scalable Vector Graphics), soporte para MathML (Math Markut Language), soporte para estilos en cascada (CSS), y y otras opciones. Una gran característica consiste en que puede ver los enlaces que se crean con el editor.

![Amaya](http://www.awven.com/salida.php?imagen=http://www.w3.org/Amaya/screenshots/screen3.png)

Sitio web: [http://www.w3.org/Amaya/](http://www.w3.org/Amaya/)

**Quanta:** Trabaja con HTML y lenguajes asociados. Es transparente con todos los lenguajes y respeta todas las normas elaboradas por la W3C con respecto a las etiquetas HTML y al código CSS. Soporta una gran cantidad de herramientas externas que ayudan en el desarrollo, entre las que se incluyen los depuradores de código. Utiliza el entorno KDE.

_**Quanta Plus**_ (Quanta+) es una herramienta libre de desarrollo de páginas web diseñado para el proyecto KDE. Su versión actual es la 3.5. Forma parte del paquete kdewebdev.

![Quanta](http://www.awven.com/salida.php?imagen=http://quanta.sourceforge.net/screenshots/snapshot02.png)

Sitio web: [http://quanta.sourceforge.net/](http://quanta.sourceforge.net/)

Existen muchos más software orientado al diseño y programación web, pero estos son los más usados por la comunidad. Si sabes o conoces de uno que hayas usado, envianos tu articulo y con gusto lo publicamos.
