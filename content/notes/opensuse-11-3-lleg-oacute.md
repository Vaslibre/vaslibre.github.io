_openSUSE es el nombre de la distribución y proyecto libre auspiciado por Novell y AMD4 para el desarrollo y mantenimiento de un sistema operativo basado en Linux. Después de adquirir SUSE Linux en enero de 2004,5 Novell decidió lanzar SUSE Linux Professional como un proyecto completamente de código abierto, involucrando a la comunidad en el proceso de desarrollo.6 La versión inicial fue una versión beta de SUSE Linux 10.0_ (fuente: [Wikipedia](http://es.wikipedia.org/wiki/OpenSUSE))

Entre estas nuevas características, openSUSE también proporciona soporte para netbooks y Btrfs el archivo de soporte del sistema. El soporte está mejorado para hardware con el kernel 2.6.34 y actualización para los controladores de gráficos.

Ofrece KDE, GNOME y Xfce, y ahora también ofrece el entorno de escritorio ligero LXDE.
